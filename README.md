# NZ Exam Timer

[jakebnz.github.io/nz-exam-timer/](https://jakebnz.github.io/nz-exam-timer/)

Simple website for displaying exam codes and a live clock for New Zealand Qualifications Authority (NZQA) exams. Displays up to 6 exam codes.

[Demo with exam codes](https://jakebnz.github.io/nz-exam-timer/?demo=6)
